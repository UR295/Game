# Rock-Paper-Scissors Game 🎮

This is a simple **Rock-Paper-Scissors** game built using [Streamlit](https://streamlit.io/). The app allows users to play the classic game against the computer, with a user-friendly interface and real-time feedback.

---

## Features ✨

- Choose between Rock, Paper, or Scissors.
- Computer randomly selects its choice.
- Instant results to show who won.
- Score tracking for multiple rounds.
- Interactive and lightweight UI.

---

## How to Run 🚀

1. **Clone the Repository**
   ```bash
   git clone https://github.com/UR295/rock-paper-scissors-streamlit.git
   cd rock-paper-scissors-streamlit
   ```

2. **Install Dependencies**
   Make sure you have Python installed. Install the required packages:
   ```bash
   pip install streamlit
   ```

3. **Run the App**
   ```bash
   streamlit run app.py
   ```

4. **Play the Game**
   I will be adding the link for the game soon..

---

## How to Play 🤔

1. Select your choice: **Rock**, **Paper**, or **Scissors**.
2. Click the "Play" button.
3. See the results instantly:
   - 🥇 **Win**: If your choice beats the computer's.
   - 🤝 **Draw**: If both make the same choice.
   - ❌ **Lose**: If the computer's choice beats yours.
4. Play again to challenge your luck and skill!
